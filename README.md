# 🏎️ Pit-Stop-Simulation

Um simulador de troca de pneus em *pixel art*, focado em agilidade e reflexos, totalmente desenvolvido na linguagem **C**.

## 🏁 Sobre o Projeto

No automobilismo de alto nível, uma corrida pode ser ganha ou perdida dentro dos boxes. O **Pit-Stop-Simulation** recria essa tensão em um formato de mini-game. O objetivo do jogador é realizar a troca completa dos pneus no menor tempo possível, exigindo coordenação e velocidade. O projeto une a lógica de programação em C com elementos gráficos retrô em *pixel art*, criando uma experiência divertida e desafiadora.

## ✨ Funcionalidades

* **Mecânica de Precisão:** Desafios de tempo real (Time Attack) para concluir o pit stop.
* **Estilo Visual Retrô:** Gráficos charmosos em *pixel art*.
* **Alta Performance:** Desenvolvido puramente em C, garantindo execução leve e rápida.
